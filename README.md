# Beekeeper Gradle Plugins

A set of plugins used in beekeeper's projects.


## Included Plugins

### `io.beekeeper.gradle`

Convention plugin for which plugins to apply.

### `io.beekeeper.formatter`

Formatting conventions for the following languages with `spotless`:

- `java`
